Synthetic Data Generator


Synthetic Data Generator is tool that allow to generate non-existing data for specific fields available on schema. The end user can upload own schema, use default one or define it directly in application. Below are presented the ways of using the application.

Note: Data Generator allow to generate unlimited volume of data, but limited type of data. Check below for available types of data.

Default Synthetic Data Generator

Default Synthetic Data Generator has 4 option for initial data input and 5 buttons (6 buttons after add more fields).

Input fields:

  • Field type - choose type of data for current column;
  • Custom name - specify custom name for current column, otherwise default value will be used;
  • Number of rows - specify number of rows to be generated;
  • File name - specify the name of the database that will be generated;

Buttons:

  • Add - add more columns in schema;
  • Remove - remove column from schema (appears after more columns were added);
  • Preview - generate temporary file and make preview of the result;
  • Download - save to local disk generated database;
  • Cloud ↑ - upload to cloud storage generated database;
  • Cloud ↓ - download from cloud storage saved database;

Available data types:

  • first_name - generate first name - Vasile;
  • last_name - generate last name - Popescu;
  • personal_number - generate social personal number - 8240804276204;
  • birthdate - generate date - 2023-04-19;
  • address - generate address with postcode - Intrarea Nr. 167 Nicolau Mare, 197205;
  • county - generate county name - Hunedoara;
  • phone_number - generate phone number - 0248 455 730;
  • mac_address - generate mac address - d4:a3:82:98:85:a6;
  • ip_address - generate IP address - 172.25.185.30;
  • job - generate job title - Pictor;
  • iban - generate bank account number - RO83YUNU3862659030037214;
  • currency - generate currency code - XAF;
  • balance - generate random number - 75564642;
  • latitude - generate latitude value - 27.6863225;
  • longitude - generate longitude value - 44.431989;

Mode of work:

  • Choose data type - choose data type for first column;
  • Add more column - press + button for number of desired columns;
  • Provide custom name - provide custom name for chosen column;
  • Remove column - remove specific column;
  • Provide number of rows - provide integer value for the number of rows to be generated;
  • Provide file name - provide the name of the database;
  • Preview - press Preview button, the result will appear in Preview section;
  • Download - press Download and download generated database on local disk;
  • Cloud ↑ - press Cloud ↑ and upload generated database in Cloud. The database will be stored in Cloud for 30 days;
  • Cloud ↓ - provide file name and press Cloud ↓ and download database from Cloud;
  • Repeat - repeat the process if necessary;

Intermediate

Final
