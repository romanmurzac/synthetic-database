Synthetic data is artificial data that is created by using Faker library that mirror the real properties of the original data but does not reveal any information regarding real people.

Synthetic Data Generator is an application that allows anyone to generate data that inherits the properties of real data, but does not represent data of real people. This allows the generated data to be used for testing any system that uses user data. Also, the data can be used to demonstrate some functionalities that use sensitive data.

Depending on your needs and level of experience, the application offers three ways of generating data, each of them having different input data, but the same output data.
